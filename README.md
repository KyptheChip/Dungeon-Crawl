# Dungeon Crawl

## Story

- Dungeon Crawl is a roguelike(or dungeon crawler) game that sets you, the Hero, inside the Dungeon of Doom. You can:
    - Move around
    - Fight enemies
    - Collect armor, weapons and food/heals
    - Use a cheatcode with the developer's name

##  Technologies used

- Git and Github
- SQL
- Java
- JavaFX
